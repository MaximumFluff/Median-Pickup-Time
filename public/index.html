Wolt Backend Assignment

The API requires three arguments: a location_id, a start_time argument and a end_time argument

The location_id requires a number between 01-99 and the other two values are Datetime strings (example: 2019-01-09T12:00:00)

The timezones can either have Z (UTC time parameter) at the end of the string or not. if it's missing it will be added to the string to ensure correct median values are returned from the CSV file

Example

http://localhost:8000//median_pickup_time?location_id=12&start_time=2019-01-09T11:00:00&end_time= 2019-01-09T12:00:00

If values are found within that range, you will recieve a JSON object with the median like so:

{ median: 27 }

Otherwise, you will recieve a JSON object containing an error message

Values are saved in the cache for 1 minute
