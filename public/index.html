The API requires three arguments: a location_id, a start_time argument and a end_time argument

The location_id is a number value and the other two values are Datetime strings (e.g 2019-01-09T12:00:00)

Example

http://localhost:8000/median_pickup_time?location_id=12&start_time=2019-01-09T20:00:00&end_time=2019-01-09T21:00:00

If values are found within that range, you will recieve a JSON object with the median like so:

{ median: 29.5 }

Otherwise, you will recieve a JSON object containing an error message
